# 로또 번호 추첨 웹앱 프로젝트

## 개요

이 프로젝트는 사용자가 버튼 클릭으로 무작위 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션을 만드는 것을 목표로 합니다. 앱은 현대적인 디자인 원칙을 사용하여 시각적으로 매력적이고 반응이 빠른 인터페이스를 제공합니다.

## 기능

*   **번호 생성:** 사용자는 버튼을 클릭하여 1부터 45까지의 숫자 중 6개의 고유한 무작위 번호를 생성할 수 있습니다.
*   **번호 표시:** 생성된 번호는 시각적으로 구분되는 스타일의 공 모양 UI에 표시됩니다.
*   **반응형 디자인:** 앱은 데스크톱 및 모바일 장치 모두에서 잘 보이도록 설계됩니다.

## 디자인 및 스타일

*   **색상 팔레트:** 다채롭고 생생한 색상을 사용하여 각 번호 공을 시각적으로 돋보이게 합니다.
*   **타이포그래피:** 가독성이 좋은 현대적인 글꼴을 사용합니다.
*   **레이아웃:** Flexbox를 사용하여 요소를 중앙에 배치하고 깔끔한 레이아웃을 유지합니다.
*   **애니메이션:** 부드러운 전환 및 그림자 효과를 추가하여 사용자 경험을 향상시킵니다.
*   **아이콘:** 의미를 명확히 하고 시각적인 흥미를 더하기 위해 아이콘을 사용합니다.

## 구현 계획

1.  **HTML (`index.html`):**
    *   페이지 제목을 "로또 번호 생성기"로 변경합니다.
    *   생성된 번호가 표시될 컨테이너를 추가합니다.
    *   번호 생성을 위한 버튼을 추가합니다.

2.  **CSS (`style.css`):**
    *   전체적인 페이지 스타일(배경, 글꼴 등)을 설정합니다.
    *   번호가 표시될 공의 스타일을 지정합니다.
    *   버튼 스타일을 지정합니다.
    *   반응형 레이아웃을 위한 미디어 쿼리를 추가합니다.

3.  **JavaScript (`main.js`):**
    *   버튼 클릭 이벤트를 감지하는 이벤트 리스너를 추가합니다.
    *   6개의 고유한 무작위 번호를 생성하는 함수를 작성합니다.
    *   생성된 번호를 HTML에 동적으로 추가하고 스타일을 적용하는 함수를 작성합니다.
